# Как включить управление через Telegram

Теперь ботом можно управлять прямо из Telegram кнопками.

## 1. Найди токен

Открой Telegram и зайди в `@BotFather`.

Напиши:

```text
/mybots
```

Выбери своего бота.

Нажми:

```text
API Token
```

Скопируй токен. Он выглядит примерно так:

```text
123456789:AAExampleTokenExampleToken
```

## 2. Добавь токен в Railway

Открой проект на Railway.

Зайди в:

```text
Variables
```

Добавь переменную:

```text
TELEGRAM_BOT_TOKEN
```

Значение — токен от BotFather.

Еще добавь переменную:

```text
PUBLIC_URL
```

Значение — ссылка Railway на твой сайт, например:

```text
https://your-project.up.railway.app
```

## 3. Перезапусти Railway

Нажми:

```text
Redeploy
```

## 4. Открой своего бота в Telegram

Напиши:

```text
/start
```

Бот покажет кнопки:

- Аккаунты
- Запустить
- Статистика
- Стоп
- Помощь

## Что делает кнопка Аккаунты

Она сразу показывает тестовые аккаунты:

```text
@test_bags_0001 | bags | 1200 подписчиков | сайт: да
@test_jewelry_0002 | jewelry | 850 подписчиков | сайт: нет
```

Это безопасные тестовые аккаунты, не реальные Instagram-профили.
