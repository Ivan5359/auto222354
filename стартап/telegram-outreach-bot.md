# Telegram outreach bot

Это полуавтоматический Telegram-бот для холодного outreach без авторассылки в Instagram.

Он делает:
- хранит лиды;
- фильтрует по подписчикам, нише и наличию сайта;
- показывает следующий аккаунт;
- даёт кнопку открыть Instagram;
- даёт готовый текст;
- хранит, кому уже писали;
- ведёт статусы: новый, написали, ответил, интересно, не подходит;
- показывает follow-up через заданное время;
- держит дневной лимит.

Он не делает:
- не логинится в Instagram;
- не сканирует Direct;
- не отправляет сообщения автоматически.

## Запуск без PowerShell

1. Дважды нажми:

```text
start_telegram_bot.bat
```

2. Окно должно остаться открытым, пока бот работает.
3. Открой своего Telegram-бота и напиши:

```text
/start
```

Токен хранится в `telegram_bot_token.txt`.

## Запуск через команду, если нужно

1. Создай бота через `@BotFather` в Telegram.
2. Скопируй токен.
3. В PowerShell из папки проекта выполни:

```powershell
$env:TELEGRAM_BOT_TOKEN="сюда_токен_бота"
python telegram_outreach_bot.py
```

База сохранится в файле `outreach_bot.sqlite3`.

## Быстрый сценарий

В Telegram напиши боту:

```text
/start
```

Добавь лиды:

```text
/add
```

Потом отправь список:

```text
@brand.ua | 1200 | сумки | no-site | только Instagram
@jewelry.ua | 3400 | украшения | has-site | сайт слабый
@linen.ua | 800 | одежда | no-site | хороший визуал
```

Настрой фильтры:

```text
/filter min=500 max=5000 site=no_site niche=сумки
```

Получить следующий профиль:

```text
/next
```

Дальше:
1. нажимаешь `Открыть Instagram`;
2. копируешь текст;
3. отправляешь вручную;
4. нажимаешь `Написал`;
5. бот сразу даёт следующий лид.

## Команды

```text
/start
/help
/add
/next
/filter
/filter min=500 max=5000 site=no_site niche=сумки
/template
/followup_template
/followups
/limit 20
/stats
/reset_skipped
```

## Формат лидов

```text
@handle | подписчики | ниша | no-site/has-site | заметка
```

Примеры:

```text
@episodebags.ua | 2400 | сумки | no-site | пример
@warmjewelry.ua | 1.3k | украшения | no-site | только Direct
@softcandles.ua | 4 тыс | свечи | has-site | сайт можно улучшить
```

## Как менять текст

Команда:

```text
/template
```

Потом отправь новый текст одним сообщением.

Можно использовать переменные:

```text
{handle}
{niche}
{followers}
{site_status}
```

Пример:

```text
Здравствуйте! Увидел ваш бренд {handle}. Мне кажется, вам могла бы помочь простая страница для клиентов: товары, доставка, контакты и кнопка заказа в одном месте.
```

## Follow-up

Команда:

```text
/followups
```

Бот покажет аккаунты, которым писали больше 48 часов назад и по которым ещё не было ответа.
